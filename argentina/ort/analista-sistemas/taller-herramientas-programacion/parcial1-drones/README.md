## Qué vas a aprender
- **Ciclos combinados:** Uso de un `for` anidado con un `do-while`.

- **Acumuladores y Contadores:** Procesamiento de estadísticas por ciudad y totales globales.
- **Búsqueda de Máximos:** Identificación del elemento mayor basado en una condición.
- **Casting y Promedios:** Manejo de tipos de datos para cálculos precisos.
