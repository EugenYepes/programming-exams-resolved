# Exámenes de Programación Resueltos (Java, C, POO, Algoritmos)

Repositorio con resoluciones de exámenes reales de programación (principalmente ORT y UTN), enfocado en entender **cómo encarar los ejercicios y aprobar**.

---

## Objetivo

Este repositorio no busca solo mostrar soluciones, sino ayudarte a:

* Entender cómo modelar problemas
* Aplicar correctamente POO (herencia, polimorfismo, encapsulamiento)
* Resolver ejercicios típicos de parciales y finales
* Identificar errores comunes que hacen desaprobar

---

## Estructura del repositorio

Los exámenes están organizados por:

```
/programming-exams-resolved/{pais}/{institucion}/{materia}/{examen}
```

### Ejemplo

```
/programming-exams-resolved/argentina/ort/tp1/examen-biblioteca
/programming-exams-resolved/argentina/utn/info1-electronica/examen-listas
```

Cada examen contiene:

* `enunciado/` → consigna original
* `starter/` → código base (si aplica)
* `solucion/` → implementación completa
* `README.md` → explicación del enfoque

---

## Qué vas a encontrar

## Importante

Leer la solución no alcanza para aprobar.

Lo importante es entender:

* por qué está modelado así
* cómo aplicar esas ideas en otros ejercicios

3. Revisá la solución
4. Mirá la explicación (si está disponible)
